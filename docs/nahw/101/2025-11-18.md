This session focused on Mosul Sifa (adjectives and their relationship to described nouns) and introduced Al-Amr (command form). ---

## Grammar Rules Covered

### Mosul Sifa (Adjective-Noun Relationship)
1. **Word Order**: Mawsuf (described noun) comes first, then Sifa (adjective) - opposite of English
2. **Four Agreements Required**:
   - Number (singular/dual/plural)
   - Gender (masculine/feminine) 
   - I'rab (rafa'/nasab/jar case)
   - Ma'rifa/Nakira (definite/indefinite status)
3. **Sifa follows Mawsuf**: The adjective must match the noun, not vice versa

### Al-Amr (Command Form)
1. **Only Second Person**: Commands are only given to the person in front of you (6 forms: 3 masculine, 3 feminine)
2. **Three Steps to Form Amr**:
   - Step 1: Give jazm (sukun) to Fi'l Mudari' / Drop Nun al-Wiqayah if present
   - Step 2: Drop Harf Mudari' (ta/ya/alif/nun prefix)
   - Step 3: Add hamza at beginning (since Arabic words cannot start with sukun)

### Hamza Placement Rules in Amr
- If Ayn Kalima (middle letter) of Fi'l Mudari' has fatha or kasra → Hamza gets kasra
- If Ayn Kalima of Fi'l Mudari' has damma → Hamza gets damma

## Important Vocabulary

| Arabic Term | Meaning |
|---|---|
| Sifa | Adjective (alternative term: Na'at) |
| Mawsuf | Described noun (alternative term: Man'ut) |
| Na'at | Adjective (alternative term for Sifa) |
| Man'ut | Described noun (alternative term for Mawsuf) |
| Al-Amr | Command form |
| Harf Mudari' | Present tense prefix letters (alif, nun, ya, ta) |
| Ayn Kalima | Middle letter/root letter of the verb |
| Nun al-Wiqayah | Protective nun (dropped in jazm/nasab cases) |
| Jazm | Grammatical state requiring sukun ending |
| Hamza | Glottal stop letter added to beginning of Amr verbs |

## I'rab (Sentence Analysis) Examples

### Mosul Sifa Examples:
- **Mahmudun al-maliku** - "Mahmud the king" (both masculine, singular, definite, rafa')
- **Tariqun al-qa'idu** - "Tariq the commander" (both masculine, singular, definite, rafa')
- **Abdun salihun** - "A pious slave" (both masculine, singular, indefinite, rafa')
- **Al-abdus salihu** - "The pious slave" (both masculine, singular, definite, rafa')

### Divine Names Examples:
- **Bismillahi ar-Rahman ar-Rahim** - Allah is mawsuf, ar-Rahman and ar-Rahim are two sifat

## Practice Examples

### Amr Formation Examples:
- **Taf'alu → If'al** (you do → do!)
- **Taftahu → Iftah** (you open → open!)
- **Tajlisu → Ijlis** (you sit → sit!)
- **Tasma'u → Isma'** (you hear → hear!)
- **Tadribu → Idrib** (you hit → hit!)
- **Tadhabu → Idhab** (you go → go!)

### Complete Amr Conjugation Pattern:
- If'al, If'ala, If'alu, If'ali, If'ala, If'alna

## Key Takeaways

1. **Proper nouns are inherently definite** - no need for al-iflam (Mahmud is already definite)
2. **Divine names rules**: Allah's descriptive names (sifati) can be used for humans without al-iflam, but essential names (zati) like Allah and possibly Rahman cannot be used
3. **Amr requires memorization** - The six-form pattern (If'al, If'ala, If'alu, If'ali, If'ala, If'alna) must be practiced repeatedly
4. **Hamza voweling is systematic** - Based on the middle letter's vowel in the original Fi'l Mudari'
