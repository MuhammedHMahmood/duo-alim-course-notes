## How These Notes Are Made

These notes are generated through an automated pipeline:

1. **Recording** — Class recordings are uploaded to Google Drive by the course coordinators
2. **Transcription** — Audio is transcribed using [OpenAI Whisper](https://github.com/openai/whisper) (large-v3-turbo model) with local GPU acceleration
3. **Note Generation** — Transcripts are processed by [Claude](https://claude.ai) (Anthropic) to produce structured study notes following subject-specific templates
4. **Publishing** — The site is built with [MkDocs](https://www.mkdocs.org/) using the [Material](https://squidfunk.github.io/mkdocs-material/) theme and hosted on GitHub Pages

Each session's notes include a session overview, key themes, detailed explanations, Arabic vocabulary with transliterations, and practical takeaways. Notes are AI-generated and may contain inaccuracies — always verify against primary sources and your own class notes.
